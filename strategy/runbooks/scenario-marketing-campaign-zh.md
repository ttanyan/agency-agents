# 📢 运行手册：多渠道营销活动

> **模式**：NEXUS-Micro 到 NEXUS-Sprint | **持续时间**：2-4周 | **代理**：10-15个

---

## 场景

你正在跨多个渠道启动协调的营销活动。内容需要针对平台特定，品牌一致，并且数据驱动。活动需要推动可衡量的用户获取和参与度。

## 代理名单

### 活动核心
| 代理 | 角色 |
|-------|------|
| 社交媒体策略师 | 活动负责人，跨平台策略 |
| 内容创建者 | 所有格式的内容制作 |
| 增长黑客 | 获取策略，漏斗优化 |
| 品牌守护者 | 所有渠道的品牌一致性 |
| 分析报告员 | 性能跟踪和优化 |

### 平台专家
| 代理 | 角色 |
|-------|------|
| Twitter 互动专家 | Twitter/X 活动执行 |
| TikTok 策略师 | TikTok 内容和增长 |
| Instagram 策展人 | Instagram 视觉内容 |
| Reddit 社区建设者 | Reddit 真实互动 |
| 应用商店优化师 | 应用商店存在（如果是移动应用） |

### 支持
| 代理 | 角色 |
|-------|------|
| 趋势研究员 | 市场时机和趋势对齐 |
| 实验跟踪器 | A/B 测试活动变体 |
| 执行摘要生成器 | 活动报告 |
| 法律合规检查器 | 广告合规，披露要求 |

## 执行计划

### 第1周：策略与内容创建

```
第1-2天：活动策略
├── 社交媒体策略师 → 跨平台活动策略
│   ├── 活动目标和KPI
│   ├── 目标受众定义
│   ├── 平台选择和预算分配
│   ├── 内容日历（4周计划）
│   └── 每个平台的参与策略
│
├── 趋势研究员 → 市场时机分析
│   ├── 要对齐的热门话题
│   ├── 竞争对手活动分析
│   └── 最佳启动时机
│
├── 增长黑客 → 获取漏斗设计
│   ├── 着陆页优化计划
│   ├── 转化漏斗映射
│   ├── 病毒机制（推荐，分享）
│   └── 渠道预算分配
│
├── 品牌守护者 → 活动品牌指南
│   ├── 活动特定的视觉指南
│   ├── 消息框架
│   ├── 活动的语调和声音
│   └── 注意事项
│
└── 法律合规检查器 → 广告合规审查
    ├── 披露要求
    ├── 平台特定的广告政策
    └── 监管约束

第3-5天：内容制作
├── 内容创建者 → 多格式内容创建
│   ├── 博客文章 / 文章
│   ├── 电子邮件序列
│   ├── 着陆页文案
│   ├── 视频脚本
│   └── 社交媒体文案（平台适配）
│
├── Twitter 互动专家 → Twitter特定内容
│   ├── 启动线程（10-15条推文）
│   ├── 日常互动推文
│   ├── 回复模板
│   └── 标签策略
│
├── TikTok 策略师 → TikTok内容计划
│   ├── 视频概念（3-5个视频）
│   ├── 钩子策略
│   ├── 流行音频/格式对齐
│   └── 发布时间表
│
├── Instagram 策展人 → Instagram内容
│   ├── 动态帖子（轮播，单图）
│   ├── Stories内容
│   ├── Reels概念
│   └── 视觉美学指南
│
└── Reddit 社区建设者 → Reddit策略
    ├── 子版块定位
    ├── 价值优先的帖子草稿
    ├── 评论互动计划
    └── AMA准备（如适用）
```

### 第2周：启动与激活

```
第1天：预启动
├── 所有内容已排队和安排
├── 分析跟踪已验证
├── A/B测试变体已配置
├── 着陆页已上线并测试
└── 团队已了解参与协议

第2-3天：启动
├── Twitter 互动专家 → 启动线程 + 实时互动
├── Instagram 策展人 → 启动帖子 + 故事
├── TikTok 策略师 → 启动视频
├── Reddit 社区建设者 → 真实社区帖子
├── 内容创建者 → 博客文章发布 + 电子邮件推送
├── 增长黑客 → 付费活动激活
└── 分析报告员 → 实时仪表板监控

第4-5天：优化
├── 分析报告员 → 前48小时性能报告
├── 增长黑客 → 基于数据的渠道优化
├── 实验跟踪器 → A/B测试早期结果
├── 社交媒体策略师 → 参与策略调整
└── 内容创建者 → 基于反馈的响应内容
```

### 第3-4周：维持与优化

```
每日：
├── 平台代理 → 互动和内容发布
├── 分析报告员 → 每日性能快照
└── 增长黑客 → 漏斗优化

每周：
├── 社交媒体策略师 → 活动性能审查
├── 实验跟踪器 → A/B测试结果和新测试
├── 内容创建者 → 基于性能数据的新内容
└── 分析报告员 → 每周活动报告

活动结束：
├── 分析报告员 → 综合活动分析
├── 增长黑客 → ROI分析和渠道有效性
├── 执行摘要生成器 → 活动执行摘要
└── 社交媒体策略师 → 经验教训和建议
```

## 活动指标

| 指标 | 目标 | 负责人 |
|--------|--------|-------|
| 总覆盖范围 | [基于预算的目标] | 社交媒体策略师 |
| 参与率 | 跨平台平均 > 3% | 平台代理 |
| 点击率 | CTA > 2% | 增长黑客 |
| 转化率 | 着陆页 > 5% | 增长黑客 |
| 获客成本 | < [目标CAC] | 增长黑客 |
| 品牌情感 | 净正面 | 品牌守护者 |
| 发布的内容数量 | [目标数量] | 内容创建者 |
| 完成的A/B测试 | ≥ 5 | 实验跟踪器 |

## 平台特定KPI

| 平台 | 主要KPI | 次要KPI | 代理 |
|----------|------------|---------------|-------|
| Twitter/X | 展示次数 + 参与率 | 关注者增长 | Twitter 互动专家 |
| TikTok | 观看次数 + 完成率 | 关注者增长 | TikTok 策略师 |
| Instagram | 覆盖范围 + 保存数 | 个人资料访问 | Instagram 策展人 |
| Reddit | 点赞数 + 评论质量 | 推荐流量 | Reddit 社区建设者 |
| 电子邮件 | 打开率 + CTR | 退订率 | 内容创建者 |
| 博客 | 有机流量 + 页面停留时间 | 反向链接 | 内容创建者 |
| 付费广告 | ROAS + CPA | 质量得分 | 增长黑客 |

## 品牌一致性检查点

| 检查点 | 时间 | 代理 |
|-----------|------|-------|
| 发布前内容审查 | 每个内容 | 品牌守护者 |
| 视觉一致性审计 | 每周 | 品牌守护者 |
| 声音和语调检查 | 每周 | 品牌守护者 |
| 合规审查 | 启动前 + 每周 | 法律合规检查器 |
